# Data Analysis & Machine Learning Coursework

**EN:**
This public repository contains my practice notebooks from the *Data Analysis and Machine Learning* course at Jahad Daneshgahi (Tehran). All exercises are implemented in Python using Jupyter Notebook and cover core topics such as Numpy, Pandas, Data Visualization, Statistics, Preprocessing, and basic Machine Learning algorithms.

**FA:**
این ریپو شامل تمرین‌های من از دوره‌ی *تحلیل داده و یادگیری ماشین* جهاد دانشگاهی تهران است. تمام تمرین‌ها با پایتون و جوپیتر نوت‌بوک نوشته شده‌اند و موضوعاتی مثل نامپای، پانداس، مصورسازی داده، آمار، پیش‌پردازش و الگوریتم‌های یادگیری ماشین را پوشش می‌دهند.
